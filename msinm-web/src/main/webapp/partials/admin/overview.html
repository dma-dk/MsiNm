Overview

The Admin pages are used to administer users, groups and base data for the MSI-NM application. It has been divided into the following sections:

Operations
Various operations pertaining to the running system.
Users
Create, update and manage users registered for the MSI-NM website.
Legacy Integration
Handle integration with the legacy MSI system and import historical NM PDF's.
Charts
Configure the OpenLayers charts used for the MSI-NM website.
Areas
Manage the list of areas used for grouping MSI and NM messages.
Categories
Manage the list of categories used for grouping MSI and NM messages.
