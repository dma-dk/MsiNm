Integration

MSI and NM warnings can be accessed via a public JSON-API, as exemplified below: 

$.getJSON(
  'http://theserver/rest/message/search?status=ACTIVE',
  function(data) {
    $('#msiList').html(
       '<h2>Behold my ' + data.total + ' messages</h2>'
     + '<p>' + JSON.stringify(data.messages) + '</p>');
  });

Furthermore, the API contains functionality that may be used by registered users.
