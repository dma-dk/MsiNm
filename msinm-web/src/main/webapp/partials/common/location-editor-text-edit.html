Edit locations

Edit the locations as plain text. The format is as follows:

  • Each location must start with a "Type: " field.
  • Circles must specify a "Radius: " field.
  • The location may have a description fields starting with "[language id]:"
  • The points have the format: 56 06.993N 007 33.018E
  • Points may have descriptions associated by appending ",[language id]: description" to the lines.
